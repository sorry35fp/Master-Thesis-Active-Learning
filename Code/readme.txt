Before you start using any code, please read the chapter "Experimental Setup" from the thesis and go through the 
"General.ipynb" to understand the code better.

The purpose of "General.ipynb" is to give a overview of the methods and to explain the different functions.

To run the code for fastText, download the vectors from here: https://dl.fbaipublicfiles.com/fasttext/vectors-crawl/cc.en.300.bin.gz
As this file is about 7 GB, we cannot upload it to GitHub
